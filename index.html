Introducción

MCreator Web Lite Modular es una herramienta gratuita que funciona 100% en el navegador. No necesitas instalar nada: abre el archivo HTML y empieza a crear elementos de mods para Minecraft.

Características principales:

  • Todo en un solo archivo HTML (como un cuaderno).
  • Dos modos: No Programar (visual, fácil para principiantes) y Programar (avanzado para desarrolladores).
  • Crea bloques, items, mobs, armaduras, GUI, recetas, comandos, procedimientos, dimensiones, biomas y estructuras.
  • Sube texturas y modelos directamente.
  • Exporta un ZIP con todo lo creado (JSON descriptivos + archivos subidos).

Requisitos: Solo un navegador moderno (Chrome, Edge, Firefox). Funciona en PC, tablet y móvil.

Modos de Uso

Al abrir el archivo HTML, verás dos botones grandes:

  • Modo No Programar (Visual): Ideal para principiantes. Todo se hace eligiendo opciones, rellenando campos y subiendo archivos. No escribes código.
  • Modo Programar (Avanzado): Para desarrolladores. Tienes editores de texto para escribir JSON y código Java directamente.

Haz click en el modo que prefieras. Puedes cambiar de modo cuando quieras.

Recomendado empezar con Modo No Programar para aprender cómo funciona cada elemento.

Modo No Programar (Visual)

Este modo es el más fácil. Solo usas dropdowns, campos de texto y botones para subir archivos. Todo se genera automáticamente.

Cómo usar una pestaña (ejemplo Bloques):

  1. Elige la pestaña (Bloques, Items, Mobs, etc.).
  2. Rellena los campos: nombre, propiedades (dureza, luz, material...).
  3. Sube una textura o modelo (PNG, JSON...).
  4. Verás una previsualización de la textura.
  5. Haz click en Generar → Aparece un JSON descriptivo con todo lo que configuraste.

Pestañas disponibles y qué hacen:

  • Bloques: Nombre, material, dureza, luz, textura. Genera bloque con modelo automático.
  • Items: Nombre, tipo (herramienta, comida...), stack, durabilidad, textura.
  • Mobs: Nombre, vida, daño, comportamiento, textura/modelo.
  • Armaduras: Nombre, slot, material, efectos, textura.
  • GUI: Nombre, slots, tipo, textura de fondo.
  • Recetas: Tipo (crafteo/fundición), grid 3x3 visual, resultado.
  • Comandos: Nombre, permiso, acción predefinida, parámetros.
  • Procedimientos: Nombre, trigger (al romper bloque, interactuar...), acción (dar efecto, spawnear mob...).
  • Dimensiones: Nombre, tipo (Overworld/Nether/End), color.
  • Biomas: Nombre, tipo, color de hierba/agua.
  • Estructuras: Nombre, tipo, sube archivo NBT.

Todo lo que generas se guarda en memoria. Al exportar, se incluye todo en el ZIP.

Modo Programar (Avanzado)

Para usuarios con conocimientos de modding. Cada pestaña tiene editores de texto para escribir JSON y código Java directamente.

Cómo usar:

  1. Elige pestaña (Bloques, Items, Entidades, Menús, etc.).
  2. Edita los campos de texto (JSON blockstate/model, código Java).
  3. Sube texturas si necesitas.
  4. Haz click en Generar → Verás un resumen con todo.

Pestañas incluyen las mismas que en No Programar + más técnicas (entidades personalizadas, screen handlers, etc.).

El ZIP exportado contiene archivos listos para un proyecto Forge/Fabric (src/main/java, resources...).

Exportar el Mod

En la pestaña "Exportar" de cualquier modo:

  • Elige versión de Minecraft y modloader (Forge/Fabric).
  • Haz click en Descargar ZIP.
  • El ZIP contiene:
    • JSON descriptivos de todos los elementos creados.
    • Texturas/modelos que subiste.
    • En Modo Programar: archivos Java y estructura de proyecto.

En iPhone/iPad o algunos móviles: Si no descarga automáticamente, mantén presionado el botón y elige "Descargar archivo vinculado" o "Guardar en Archivos".

El ZIP es una base perfecta para importar en el MCreator oficial (desktop) o continuar en un proyecto Forge/Fabric.

Preguntas Frecuentes (FAQ)

  • ¿Necesito internet? No. El archivo HTML funciona offline una vez guardado.
  • ¿Funciona en móvil? Sí. Todo responsive. En iOS/Android abre el HTML con el navegador.
  • ¿Puedo guardar progreso? No permanentemente (es web). Usa "Exportar" para guardar todo en ZIP.
  • ¿Es compatible con MCreator oficial? Sí. Los JSON son descriptivos y fáciles de recrear en MCreator desktop.
  • ¿Puedo añadir más elementos? El código es abierto. Edita el HTML si sabes JavaScript.
  • Problemas comunes: Si una pestaña no abre, recarga la página. Usa Chrome/Edge para mejor compatibilidad.
